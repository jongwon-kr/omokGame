# 원격통신 및 제어 프로그램
## 프로젝트 설명
PC에 문제가 생겨 원격으로 PC조작이 필요한 상황처럼 PC를 원격에서 이용해야 되는 상황이 발생하게 되었을 때 사용자에게 네트워크를 통해 원격으로 PC를 조작하고 파일을 전송하고 받을 수 있고 음성 통화를 지원하기 위해 제작 되었다.

## 서비스 내용
- 로그인 및 고유 키 발급
- 원격 제어
- 음성 통화
- 파일 전송

## 프로젝트 기간
2022.10 ~ 2022. 12

## 개발 환경 및 언어
- Eclipse
- Java

## 주 활용 클래스
