# 1:1 매칭 및 인공지능 대결 오목게임
## 프로젝트 설명
클라이언트-서버 통신을 통한 1:1 대결, 인공지능 컴퓨터와 대결 서비스를 구현하기 위해 제작

## 서비스 내용
- 오목게임
- 사용자 1:1 대결
- 인공지능 컴퓨터와 대결

## 프로젝트 기간
2022.01 ~ 2022.03

## 개발 환경 및 언어
- Eclipse
- Java

## 주 활용 클래스
- Swing
